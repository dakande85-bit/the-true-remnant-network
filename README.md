# The True Remnant Network

A simple MVP website for a verified Christian content and ministry network.

The platform is designed to help people discover faithful Christian teaching, churches, ministries, missionaries, charities, books, music, events and testimonies in one place.

## Core principle

This is **not** a payment marketplace. The website links out to official ministry, donation, event, book, music and clothing pages.

## MVP features

- Podcast hub for YouTube episodes and guest profiles
- Verified Christian directory
- Ministry and organisation profile pages
- Resources section with tabs for books, music, audio teaching, devotionals and Bible tools
- Map-ready structure for churches, ministries and missions
- Missions/support page using external links only
- Events listing
- Clothing page for future Christian clothing links
- Submit Ministry application flow
- Verification explanation page
- Admin dashboard placeholder

## Tech stack

- Next.js App Router
- TypeScript
- Static seed data first
- Supabase-ready data model later
- Vercel-ready deployment

## Brand

**The True Remnant**

Possible positioning:

> Find faithful voices. Discover trusted ministries. Support real Kingdom work.

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
