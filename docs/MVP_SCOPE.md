# MVP Scope — The True Remnant Network

## What this first version is

A simple Christian network website for:

- Podcast episodes
- Reviewed directory profiles
- Book reading lists
- Music/audio recommendations
- Audio teaching links
- Devotionals
- Bible tools
- Events
- Missions and external support links
- Clothing links

## What this first version is not

- No payment marketplace
- No crowdfunding processing
- No user reviews
- No open public posting
- No automatic verification
- No claim that every listed person is permanently endorsed

## MVP build priorities

1. Make the website visually strong.
2. Add real seed content manually.
3. Add Supabase tables.
4. Add admin approval workflow.
5. Add map integration.
6. Add YouTube embeds.
7. Add external links for books, music, missions and clothing.

## Verification wording

Use:

> Reviewed and listed after public information, doctrine, leadership and activity checks where possible.

Avoid:

> The only true people of God.
> Every unlisted church is false.
> Fully safe forever.

The network should be firm, biblical and careful.
