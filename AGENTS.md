# Remnant Platform Instructions

Build The True Remnant Network as a premium Christian media and trust-network platform.

## Product Identity

The True Remnant Network helps people discover validated Christian teachers, churches, ministries, charities, missionaries, worship artists, books, podcasts, events, and mission projects.

## Design Direction

- Premium Christian media platform
- Editorial and cinematic
- Clean, modern, trustworthy
- Not old-fashioned church design
- Strong typography
- Warm neutral colours: cream, black, off-white, and gold accent
- Mobile-first

## MVP Rules

- Do not add payments yet.
- Use external links for donations, YouTube, websites, books, music, and events.
- Use mock/example data until Supabase is configured.
- Keep data access isolated so Supabase can replace the mock provider cleanly.
- Keep the MVP simple and production-ready.

## Required Pages

- Home
- About
- Directory
- Organisation profile
- Person profile
- Podcast / media
- Missions
- Events
- Submit organisation
- Admin dashboard

## Required Components

- Header
- Footer
- Hero
- DirectoryCard
- ProfileCard

## Validation Statuses

- Listed
- Under Review
- Reviewed
- Verified
- Featured
